## ⚙️ Requisitos Técnicos
- Node.js v18+
- Expo SDK 49
- Dispositivo móvil o emulador (Android/iOS)

## 🛠️ Instalación
```bash
# Clonar repositorio
git clone https://github.com/tu-usuario/gestor-piezas.git

# Instalar dependencias
npm install

# Iniciar proyecto
npx expo start
